# Sales Performance & Profitability Dashboard

## Overview
This project is my first Business Intelligence dashboard built in **Power BI** connected to a **MySQL** database.  
The goal was to explore sales data and analyze revenue, profit, and profit margins across different categories, countries, and sales teams.

It helped me practice:
- SQL data integration  
- Data modeling with a Calendar table  
- KPI creation with DAX  
- Interactive dashboard design for business insights

---

## Tools Used
- Power BI  
- MySQL  
- DAX (Data Analysis Expressions)  
- Excel (for data verification)  

---

## Key Features
- Key metrics: **Total Revenue**, **Total Profit**, **Profit Margin %**, **Total Orders**  
- Sales performance breakdown by **country**, **category**, **device type**, and **sales team**  
- Interactive filtering for dynamic insights  
- Clear and intuitive visualizations

---

## Learnings
- Building a **relational data model** for BI dashboards  
- Creating **calculated measures**   
- Designing **business-oriented KPIs**  
- Presenting data clearly for decision-making  

---

## How to Use
1. Open the `sales_Dashboard.pbix` file in Power BI Desktop.  
2. Connect to your local MySQL database if needed.  
3. Explore the interactive dashboard for different dimensions.
